# Dataset Choice

Place the chosen public customer support dataset here and document it.

Requirements:
- 100+ real docs across categories (FAQs, policies, troubleshooting)
- Markdown or HTML preferred
- Public license (MIT, CC-BY, etc.)

Examples to consider (pick one and document the exact source/commit):
- GitLab Docs
- Shopify Help Center
- Notion Help Center

Ingestion: run `python -m tools.ingest` after placing files.
